📖 **Cursus:** dB, dBm, logaritmen & RF-interpretatie voor radioamateurs

---

## Overzicht

Volledig meertalig (🇧🇪 NL / 🇫🇷 FR / 🇬🇧 EN) leertraject voor radioamateurs — van absolute beginner tot HAREC-niveau.

### Inhoud (30 hoofdstukken)
| Deel | Inhoud |
|---|---|
| 0 | Oriëntatie |
| I | Wiskundige fundamenten: verhoudingen, logaritmen |
| II | De decibel: definitie, rekenen, ketens |
| III | dBm & eenheden: omrekenen, ankerpunten |
| IV | Antenneversterking: dBi, dBd, dBc |
| V | Toepassingen: kabelverlies, S-niveaus, link budget |
| VI | Meten & interpreteren, veelgemaakte fouten |
| VII | Synthese, oefenreeks, eindtest |

### Volledig uitgewerkt
- H01 Verhoudingen · H02 Logaritmen · H04 Wat is dB?
- H10 dBm · H11 Omrekenen (met interactieve calculator)
- H21 Ontvangst & S-niveaus · H26 Veelgemaakte fouten
- Printbare spiekfiche (A4)
