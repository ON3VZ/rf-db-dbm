📐 Vermogensverhouding in dB
N (dB) = 10 · log₁₀ (P₁ / P₂)
N het aantal decibel — het resultaat
P₁ het te beschrijven vermogen — teller
P₂ de referentie (ingang, vergelijkingspunt) — zelfde eenheid als P₁!
10 factor 10 geldt ALLEEN voor vermogen (voor spanning: 20)

02Stap-voor-stap methodeMéthode étape par étapeStep-by-step method

Stap 1: Controleer of P₁ en P₂ dezelfde eenheid hebben (beide W, of beide mW). Stap 2: Bereken P₁÷P₂. Stap 3: Neem log₁₀. Stap 4: Vermenigvuldig met 10. Stap 5: Interpreteer (positief = versterking, negatief = verlies, 0 = gelijk).

03Terugrekenen van dB naar vermogensverhoudingReconversion dB → rapportConverting back dB → power ratio

📐 dB → lineaire verhouding
P₁/P₂ = 10 ^ (N / 10)
voorbeeld +20 dB → 10^(20/10) = 10² = 100 → factor 100
voorbeeld −10 dB → 10^(−1) = 0,1 → factor 0,1 = ÷10
⚠️ Schrijffout

Schrijf altijd N(dB) = 10·log₁₀(P₁/P₂) — met beide termen! "dB = 10·log(P)" zonder noemer is onvolledig en inhoudelijk fout.

OefeningenExercicesExercises
5.1
●●●
P₁ = 40 W, P₂ = 10 W. Hoeveel dB?
N = 10·log₁₀(40/10) = 10·log₁₀(4) ≈ 10·0,602 = +6,02 dB ≈ +6 dB
5.2
●●●
Versterker: +17 dB versterking. Ingang 2 W. Uitgang?
+17 dB = +10+7 = ×10×5 = ×50 → P_uit = 2 × 50 = 100 W
5.3
●●●
Van 50 W naar 5 W. Hoeveel dB verlies?
N = 10·log₁₀(5/50) = 10·log₁₀(0,1) = 10·(−1) = −10 dB
5.4
●●●
Een filter heeft −23 dB. Ingang 100 mW. Uitgang in mW?
Factor = 10^(−23/10) = 10^(−2,3) ≈ 0,005 → 100 × 0,005 = 0,5 mW

Samenvatting

  • N(dB) = 10·log₁₀(P₁/P₂) — altijd met beide termen, zelfde eenheid
  • Positief = versterking · Negatief = verlies · 0 dB = gelijk
  • Terugrekenen: factor = 10^(N/10)
🧠 Onthoud dit
N = 10·log₁₀(P₁/P₂)
Terug: factor = 10^(N/10)
