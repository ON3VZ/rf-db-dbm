dBc (decibel relative to carrier) is een relatieve maat die aangeeft hoe sterk of zwak een bijkomende frequentiecomponent (harmonische, spurious) is ten opzichte van de gewenste draaggolf. Hoe negatiever, hoe beter.

dBc (decibel relative to carrier) is een relatieve maat die aangeeft hoe sterk of zwak een bijkomende frequentiecomponent (harmonische, spurious) is ten opzichte van de gewenste draaggolf. Hoe negatiever, hoe beter.

📐 Definitie dBc
P (dBc) = P_spur (dBm) − P_draaggolf (dBm)
altijd negatief spurious/harmonische is altijd zwakker dan de draaggolf
voorbeeld Draaggolf −37,73 dBm · 2e harm. −93,10 dBm → dBc = −93,10 − (−37,73) = −55,37 dBc
norm ETSI EN 301 783 / ITU-R SM.329 Cat.B: alle spurious ≤ −60 dBc (voor P ≤ 25 W)
🔑 Kernpunt dBc

dBc is onafhankelijk van de attenuatie in de meetopstelling. Als draaggolf en harmonische even sterk worden verzwakt, blijft de onderlinge verhouding (dBc) identiek. Dankzij dit principe werken attenuatoren in spectrum-analyse.

📐 dBµV ↔ dBm bij 50 Ω
P(dBm) ≈ U(dBµV) − 107 dB   [bij 50 Ω]
afleiding 0 dBm = 1 mW bij 50 Ω → U = √(0,001×50) = 0,2236 V = 223.600 µV = +107 dBµV
voorbeeld −73 dBm (S9 HF) = −73 + 107 = +34 dBµV (= 50 µV)

Samenvatting

  • dBc = P_spur(dBm) − P_carrier(dBm) · norm: ≤ −60 dBc
  • dBc is onafhankelijk van attenuatie — meting blijft geldig
  • dBµV = dB t.o.v. 1 µV · 0 dBm ≈ +107 dBµV bij 50 Ω
🧠 Onthoud dit
dBc = spur − carrier  ·  Norm: ≤ −60 dBc
dBµV ≈ dBm + 107 (50 Ω)
